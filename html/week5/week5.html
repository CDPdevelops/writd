Working with Arrays
The first task was becoming familiar and creating an array using Javascript. An array is a variable that can hold multiple values at the one time. In task 1 this array 'uni' is used to store two university names in Northern Ireland. Similarly, array 'campus' is used to store four campus names of University of Ulster and array 'course' to store four university courses. Array elements are accessed using numeric indexes (starting from 0) and can be outputted to screen using the window.document.write method. The following code is used to output to screen "I'm a third year Software Engineering student in Ulster University at Jordanstown."

Working with Functions
Task 2 consisted of creating a function called "DisplayTex (inputtext)". The function when called will display the string of 'inputtext' when it is called. The document.write method will be used to show this. The function was incorporated into a modified version of Practical4_T1.html and was placed / called within the head section. As we can see in this task the function is invoked (called) when a button is clicked located within the body.
Practical4_T1b.html Figure 2
Next, the JavaScript function could be moved to a seperate file called myjslib.js and Practical4_T1b.html could be modified, referencing an external JavaScript library. This allows fault finding easier if occurs. As we can see the function is called from the external library and triggred by the call function button.This new document was named Practical4_T1c.html.
Practical4_T1c.html Figure 3
The final task was to add a form within the Practical4_T1c.html body to gather details about the users name and what they do at university (name, uni, campus and course title). Displaying their details selected from the drop down boxes and followed by the users name incorporated into a welcome message. These details are displayed by calling the function triggred by the display details button. As we can see the first part of the function is to gather the values selected from the form and the name from the input text placeholder. The second part of the function is to return all details selected on a seperate line, using the <br> (linebreak) element. Finally the details are incorporated into a message welcoming the user to the JavaScript Homepage! The innerHTML property sets or returns the HTML content (inner HTML) of an element.
