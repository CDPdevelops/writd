Form validation
The first task was to create a multiple image roll over to display a list of your interests. For example my interests are surfing and coast adventure.
i. Create an images folder under the folder stored practical html files, for example, if you store all your html files under ‘com353’, then the new folder should be ‘com353/images/’. Skip this step, if you already have an images folder.
ii. Find 4-5 images you are going to used and save them in the images folder. If you obtain images from the internet, make sure that you respect the copyrights and reference them appropriately.
iii. Create a page with the heading image “My favourite website”, when the image is clicked, it will direct the user to your COM353 learning log index page.
Note: all images must be preload onto the web page iv. Using functions to control roll on/roll off images
v. When click on an image, the page will direct the visitor to a relevant external website.
vi. save the page as practical10_T1.html
Slideshow
Referring to class notes; create a slideshow on the web page practical10_T1.html created in T1 to display a set of relevant images.
Animated Advertising banner
Use setTimeout() to display the above images as an animated advertising banner on the webpage practical10_T1.html created in T2. When click on an image, the page will direct the visitor to a relevant external website.
