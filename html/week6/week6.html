The first step was creating the JavaScript function constructing a count-up web page similar to the one located in the practical brief, counting from 1-5. This function used similar properties to the one located in Practical 4 making use of the arithmetic operator ++, incrementing each value by 1. For the first step a for loop was used.
w3 Schools states: "The for statement creates a loop that is executed as long as a condition is true. The loop will continue to run as long as the condition is true. It will only stop when the condition becomes false"(w3 Schools).

As the i values are counting from 1-5; the var i should equal to 1 instead of 0 and should be counting to index value 6 instead of 5. Meaning the count is 1-5 and not 0-4.
This should be: for (i = 1; i < 6; i++) instead of for (i = 0; i < 5; i++).
COUNTDOWN
The next step was to modify the previous code so that instead of counting up to 5, it would be counting down from 5. Replacing the increment ++ arithmetic operator with the decrement -- operator, decreasing each value by 1.
Figure 5
Comparing Figure 1 to Figure 5, we can see that within the for loop, some arithmetic symbols and numbers alter. We can see that counting up the for loop for (i = 1; i < 6; i++), it makes use of the less than symbol '<' and the '++' operator. For the countdown, the for loop for (i = 5; i > 0; --i), makes use of the greater than symbol '>' and the '--' operator, furthermore i is equal to 5 in this case.

Further Functions
As shown above the user can select each university they attend, although when they click upon "Queens University" they should only be allowed to click the Belfast campus, as there is no Queens University campuses in Jordanstown, Magee or Coleraine. So the first simple function 'DisableUniValue()' when clicked upon the "Queens" radio button, the Jordanstown, Magee and Coleraine radio buttons will become disabled, meaning only the value Belfast can be returned. The second function 'EnableUniValue' is enabled when the 'Ulster' radio button is clicked, enabling the values that were previously disabled. As Ulster University have campuses in those locations.
The jQuery function below allows only one value to be selected within the checkboxes returning only one value. Previously without this function all values could be selected but the function displayTex() would return the first value selected. A jQuery.min.js file was referenced to googleapis.com.
