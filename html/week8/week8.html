As we can see from the html document opened in the browser. This page is used to create a multiplication table. The for (j=1; j<=3; j++) declares 3 rows of values, the for (i=1; i<=5; i++) is how many columns is in this table. The i*j multiplies the values shown on screen. This table shows the first 5 multiples of values 1, 2 and 3.
Car insurance candidate classification.
The next step was to create a web page for car insurance candidates using if statements. When a visitor inputs first name, surname, age and marital status, the web site will provide information on his/her insurance class according to the rules below:
i. if the age is above 30 (include 30) and married, the car insurance is ‘class 1’;
ii. if the age is above 30 (include 30) and single, the car insurance is ‘class 2’;
iii. if the age is below 30 and married, the car insurance is ‘class 3’;
iv. if the age is below 30, single, the car insurance is ‘class 4’.
As we can see the insurance class outputted to screen based on the rules of the form entered. Using the IF statements shown below (Figure 3).
Figure 2
Figure 3
The form was given validation by:
Giving the numnber placeholder for age only a minimum value of 17, as that is the legal age to drive.
The input placeholders for surname and first name were given the patterns pattern="[A-Za-z]{10,50}" meaning they have a minimum of 10 characters and maximum of 50 to enter uppercase and lowercase characters.
The form elements have also been assigned a required attribute, meaning the values must be selected.
